# Data Visualization Portfolio

Comprehensive collection of data visualization exercises and projects covering matplotlib, seaborn, plotly, and dash.

## Course: IBM Data Visualization with Python

This portfolio showcases skills developed through IBM's Data Visualization course, demonstrating proficiency in creating various chart types and interactive dashboards.

## Projects

### Practice Notebooks (1-9)
Progressive exercises covering:
1. Basic matplotlib plotting
2. Line charts and scatter plots
3. Bar charts and categorical data
4. Pie charts and subplots
5. Seaborn statistical visualizations
6. Plotly interactive charts
7. Geographic visualizations with Folium
8. Dashboard basics with Dash
9. Advanced dashboard interactivity

### Final Assignments
- **Assignment 1**: Comprehensive visualization project
- **Dash Tutorials**: Interactive web dashboard development

## Tech Stack

- Python 3.x
- Matplotlib - Core 2D plotting
- Seaborn - Statistical visualization
- Plotly - Interactive web-based charts
- Folium - Geographic visualizations
- Dash - Interactive dashboard framework
- Pandas - Data manipulation
- Jupyter Notebooks - Development environment

## Installation

```bash
pip install pandas numpy matplotlib seaborn plotly dash folium jupyter
```

## Notebooks

Open any notebook with:
```bash
jupyter notebook Practice_1.ipynb
```
