# MODULE 1: Introduction to Data Visualization Tools

## Why Build Visuals?
1. Exploratory data analysis
2. Communicate data clearly
3. Share unbiased representation
4. Support recommendations to stakeholders

**Best Practices:** Less is more effective, attractive, impactful

---

## Matplotlib Architecture (3 Layers)

### 1. Backend Layer
- Handles drawing via toolkits
- `FigureCanvas` = the paper
- `Renderer` = the paintbrush
- `Event` = user inputs (mouse, keyboard)

### 2. Artist Layer
- Full control of Figure
- Everything visible = Artist instance
- Types: Primitive (Line2D, Rectangle, Circle) & Composite (Axes, Figure)
- Most plotting methods defined in `Axes`

### 3. Scripting Layer (pyplot)
- Simplest interface (like MATLAB)
- Stateful - maintains current figure/axes
- Use: `import matplotlib.pyplot as plt`

---

## Plotting Basics

```python
%matplotlib inline  # render in notebook
%matplotlib notebook  # interactive (can modify after render)

import matplotlib.pyplot as plt
plt.plot(5, 5, 'o')  # plot a point
plt.show()
```

**Note:** After `plt.show()`, cannot modify. Add titles/labels BEFORE showing.

---

## Line Plots
- Best for: trend visualization over time
- `kind='line'` or just `.plot()`
- `stacked=False` for unstacked area plots

```python
haiti = df_can.loc['Haiti', years]
haiti.plot(kind='line')
plt.title('Immigration from Haiti')
plt.xlabel('Years')
plt.ylabel('Number of Immigrants')
plt.annotate('2010 Earthquake', xy=(2010, 6000))
plt.show()
```

# MODULE 2: Basic and Specialized Visualization Tools

## Area Plots
- Stacked line plot (cumulative)
- `alpha` = transparency (0-1)
- `kind='area'`

```python
df_top7.plot(kind='area', alpha=0.45, figsize=(14, 8))
```

---

## Histograms
- Frequency distribution of numerical data
- X-axis divided into bins, Y-axis = count
- Use `np.histogram()` to get bin edges

```python
count, bin_edges = np.histogram(df_can['2013'])
df_can['2013'].plot(kind='hist', xticks=bin_edges, figsize=(14, 8))
```

---

## Bar Charts
- Length = magnitude/size
- `kind='bar'` (vertical) or `kind='barh'` (horizontal)

```python
df_iceland.plot(kind='bar', figsize=(10, 6))
plt.annotate('2008 Financial Crisis', xy=(32, 70), xytext=(28, 20),
             arrowprops=dict(arrowstyle='->', color='blue'))
```

---

## Pie Charts
- Proportions as slices
- `autopct='%1.1f%%'` for percentages
- `startangle=90` for rotation
- `shadow=True` for 3D effect
- `explode` to offset slices

```python
df_continents['Total'].plot(kind='pie', autopct='%1.1f%%', 
                            startangle=90, shadow=True)
```

---

## Box Plots
**5-number summary:**
- Minimum
- Q1 (25%)
- Median (50%)
- Q3 (75%)
- Maximum

Used to detect outliers and distribution shape.

```python
df_japan.plot(kind='box', figsize=(8, 6))
df_japan.describe()  # get statistics
```

---

## Scatter Plots
- Compare variables
- Can add regression line with `np.polyfit()`

```python
df_tot.plot(kind='scatter', x='year', y='total')
x, y = df_tot['year'], df_tot['total']
fit = np.polyfit(x, y, deg=1)  # linear fit
plt.plot(x, fit[0]*x + fit[1], color='red')
```

---

## Bubble Plots
- Scatter plot with 3rd dimension (size = weight)

```python
norm_brazil = (df_can_t['Brazil'] - df_can_t['Brazil'].min()) / \
              (df_can_t['Brazil'].max() - df_can_t['Brazil'].min())

ax0 = df_can_t.plot(kind='scatter', x='Year', y='Brazil',
                    s=norm_brazil * 2000 + 10)
ax1 = df_can_t.plot(kind='scatter', x='Year', y='Argentina', ax=ax0)
```

# MODULE 4: Dashboards with Plotly and Dash

## Plotly Graph Objects vs Express

### plotly.graph_objects (Low-level)
```python
import plotly.graph_objects as go
fig = go.Figure(data=go.Scatter(x=df['x'], y=df['y'], mode='markers'))
fig.show()
```

### plotly.express (High-level)
```python
import plotly.express as px
fig = px.bar(df, x='DestState', y='Flights')
fig.show()
```

---

## Plotly Chart Types

```python
# Scatter
px.scatter(df, x='col1', y='col2')

# Bar
px.bar(df, x='category', y='value')

# Pie
px.pie(df, values='Flights', names='Category')

# Histogram
px.histogram(df, x='ArrDelay')

# Sunburst (hierarchical)
px.sunburst(df, path=['Month', 'DestState'], values='Flights')

# Bubble
px.scatter(df, x='x', y='y', size='size_col')
```

---

## Dash Basics

```python
import dash
import dash_html_components as html
import dash_core_components as dcc

app = dash.Dash(__name__)

app.layout = html.Div([
    html.H1('Dashboard Title'),
    dcc.Graph(figure=fig)
])

if __name__ == '__main__':
    app.run_server()
```

---

## Dash Callbacks (Interactivity)

```python
from dash.dependencies import Input, Output

@app.callback(
    Output('output-graph', 'figure'),
    Input('input-year', 'value')
)
def update_graph(entered_year):
    df_filtered = df[df['Year'] == int(entered_year)]
    fig = px.line(df_filtered, x='Month', y='Delay')
    return fig
```

**Key Components:**
- `@app.callback` decorator
- `Input` = triggers update
- `Output` = what gets updated
- Multiple inputs/outputs in lists

# QUICK REFERENCE: Chart Selection

| Data Type | Best Chart |
|-----------|------------|
| Trend over time | Line, Area |
| Part-to-whole | Pie, Waffle |
| Frequency distribution | Histogram |
| Compare values | Bar (vertical/horizontal) |
| Show distribution | Box Plot |
| Relationship between variables | Scatter |
| 3+ dimensions | Bubble, Sunburst |
| Geographic data | Map, Choropleth |
| Text/frequency | Word Cloud |

---

# COMMON PITFALLS

1. **Pie charts** - Generally avoid; hard to compare slices
2. **3D charts** - Distort perception, avoid
3. **Truncated Y-axis** - Can mislead
4. **Too many colors/elements** - "Less is more"
5. **Missing labels/units** - Always label axes

---

# KEY TAKEAWAYS

- Matplotlib: 3 layers (Backend, Artist, Scripting)
- Pandas has built-in `.plot()` method
- `%matplotlib inline` for notebooks
- Annotations add context (earthquakes, crises)
- Box plots show 5-number summary + outliers
- Scatter plots can show trends with regression
- Plotly/Dash for interactive dashboards
- Dash callbacks make dashboards reactive
- Folium for geospatial visualization
